PERHATIAN!

  • Sistem ini secara otomatis akan mengambil screenshot dari kamera setiap 5 detik, jika wajah terdeteksi.
  • Klik tombol Start untuk memulai dan Stop untuk menghentikan proses.
  • Pastikan kamu mengizinkan akses kamera dan download multiple files saat diminta browser.
  • Sebaiknya minimize halaman ini jika ingin ditinggal agar tampilan video kamera tidak terlihat mencolok.
  • Gambar akan terunduh otomatis ke folder Downloads laptop Anda.
  • Jika penyimpanan laptop hampir penuh, disarankan untuk mengosongkan sebagian ruang terlebih dahulu.
  • Sistem ini disarankan untuk penggunaan jangka pendek, seperti saat kamu ingin meninggalkan laptop sebentar.
  • Jika ada orang menanyakan kenapa kameranya menyala, cukup jawab sedang mengetes kamera.
  • Jangan meninggalkan laptop sendirian di tempat umum tanpa pengawasan untuk menghindari risiko pencurian!.
